# 💸 โอมจ่าย (Ohm-Jai)

> เว็บไซต์สำหรับบันทึกรายรับ - รายจ่าย ที่มีวัตถุประสงค์เพื่อ **ติดตามสถานะการเงิน สามารถแก้ไข เพิ่มเติม หรือลบข้อมูลได้อย่างรวดเร็ว**

## 📌 เกี่ยวกับโปรเจกต์ (About Project)

**โอมจ่าย (Ohm-Jai)** เป็นโปรเจกต์ที่พัฒนาโดย Arthakorn Na-rard มีเป้าหมายเพื่อสร้างแอปพลิเคชันสำหรับการบันทึกรายรับ-รายจ่าย ที่ใช้งานง่าย ไม่ซับซ้อน และช่วยให้ผู้ใช้งานสามารถมองเห็นภาพรวมทางการเงินของตนเองได้อย่างชัดเจน

แนวคิดหลักของโปรเจกต์ (Core Concept)

- ⚡ บันทึกข้อมูลได้รวดเร็ว  
- 🧾 แสดงผลข้อมูลอย่างชัดเจน
- 📊 วิเคราะห์ข้อมูลได้ง่าย
- 📱 รองรับการใช้งานที่สะดวกในชีวิตประจำวัน

---

## 🎯 วัตถุประสงค์ของโปรเจกต์

- สร้างระบบบันทึกรายรับ-รายจ่ายที่ใช้งานได้จริง
- ช่วยให้ผู้ใช้ติดตามพฤติกรรมทางการเงินของตนเอง
- พัฒนาฟีเจอร์วิเคราะห์ข้อมูลเพื่อช่วยในการวางแผนการเงิน
- ขยายความสามารถในอนาคตให้ครอบคลุมการลงทุนและระบบอัตโนมัติ

## 💡 แนวคิดสำคัญของ Ohm-Jai

> “การเงินที่ดี เริ่มจากการมองเห็นตัวเลขของตัวเองอย่างชัดเจน”

โอมจ่ายไม่ได้มุ่งเน้นเพียงแค่การบันทึกข้อมูล แต่ต้องการเป็นเครื่องมือที่ช่วยให้ผู้ใช้:
- เข้าใจพฤติกรรมการใช้เงินของตนเอง
- วางแผนการเงินได้ดีขึ้น
- ต่อยอดสู่การลงทุนในอนาคต

---

## 🛠️ เทคโนโลยีที่ใช้ (กำหนดเพิ่มเติมได้ภายหลัง)

- Front-end: Nuxt.js, Tailwind CSS
- Back-end: Nuxt.js
- Database: Supabase

## 📈 เป้าหมายระยะยาว

- เป็นเครื่องมือจัดการการเงินส่วนบุคคลที่ใช้งานง่าย
- รองรับการใช้งานบนมือถืออย่างเต็มรูปแบบ
- พัฒนาเป็น Web Application ที่สามารถใช้งานได้ทุกที่
- อาจขยายเป็น Mobile App ในอนาคต 📱
