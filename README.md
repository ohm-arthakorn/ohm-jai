# 💸 โอมจ่าย (Ohm-Jai)

> เว็บไซต์สำหรับบันทึกรายรับ-รายจ่าย ที่เน้น **ความรวดเร็ว ชัดเจน และง่ายต่อการวิเคราะห์**

---

## 📌 เกี่ยวกับโปรเจกต์

**โอมจ่าย (Ohm-Jai)** เป็นโปรเจกต์ที่พัฒนาโดยโอม มีเป้าหมายเพื่อสร้างแอปพลิเคชันสำหรับการบันทึกรายรับ-รายจ่าย ที่ใช้งานง่าย ไม่ซับซ้อน และช่วยให้ผู้ใช้งานสามารถมองเห็นภาพรวมทางการเงินของตนเองได้อย่างชัดเจน

แนวคิดหลักของโปรเจกต์คือ:

- ⚡ บันทึกข้อมูลได้รวดเร็ว
- 🧾 แสดงผลข้อมูลอย่างชัดเจน
- 📊 วิเคราะห์ข้อมูลได้ง่าย
- 📱 รองรับการใช้งานที่สะดวกในชีวิตประจำวัน

---

## 🎯 วัตถุประสงค์ของโปรเจกต์

- สร้างระบบบันทึกรายรับ-รายจ่ายที่ใช้งานได้จริง
- ช่วยให้ผู้ใช้ติดตามพฤติกรรมทางการเงินของตนเอง
- พัฒนาฟีเจอร์วิเคราะห์ข้อมูลเพื่อช่วยในการวางแผนการเงิน
- ขยายความสามารถในอนาคตให้ครอบคลุมการลงทุนและระบบอัตโนมัติ

---

## 🚀 แผนการพัฒนา (Roadmap)

### 🟢 ระยะเริ่มต้น
- [ ] พัฒนาโครงสร้างพื้นฐานของระบบ
- [ ] สร้างฟีเจอร์บันทึกรายรับ-รายจ่าย
- [ ] จัดเก็บข้อมูลลงฐานข้อมูล

### 🟡 ระยะถัดไป
- [ ] เพิ่มระบบวิเคราะห์ข้อมูลเบื้องต้น  
  - สรุปรายวัน / รายเดือน  
  - แยกตามหมวดหมู่  
  - กราฟแสดงแนวโน้มรายจ่าย  

### 🔵 ระยะพัฒนาเพิ่มเติม
- [ ] เพิ่ม Add-on ด้านการลงทุน 💰  
  - บันทึกพอร์ตการลงทุน  
  - ติดตามกำไร-ขาดทุนเบื้องต้น  

### 🟣 ระยะขั้นสูง
- [ ] รองรับการอ่าน **QR Code** 📷  
  - สแกนสลิปหรือ QR Payment  
  - บันทึกรายการรายรับ-รายจ่ายอัตโนมัติ

## 💡 แนวคิดสำคัญของ Ohm-Jai

> “การเงินที่ดี เริ่มจากการมองเห็นตัวเลขของตัวเองอย่างชัดเจน”

โอมจ่ายไม่ได้มุ่งเน้นเพียงแค่การบันทึกข้อมูล แต่ต้องการเป็นเครื่องมือที่ช่วยให้ผู้ใช้:
- เข้าใจพฤติกรรมการใช้เงินของตนเอง
- วางแผนการเงินได้ดีขึ้น
- ต่อยอดสู่การลงทุนในอนาคต
